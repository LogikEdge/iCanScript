1 # JSON Library Usage Notes
2 
3 The JSON library implements the standard JSON encoding / decoding protocol. The Library has three main APIs:
4 
5 - **static JObject JSON.GetRootObject(string JSONencodedObject):**
6  - This method decodes the string and returns the root JSON Object.
7 
8 - **JValue JObject.GetValueFor(string accessorString):**
9  - Returns the JSON value retrieve from the accessor string. JNull is return if the value cannot be determined.
10  - The accessor string format is similar to the standard language structures. A period (.) is used to access an attribute of an object and the subscribe operator ([]) is used for arrays. So to get the 3rd element from the attribute "name" of object "person" in the root object would have an accessor of:
11  - "person.name[3]"
12 
13 - **string JObject.Encode():**
14 
15  - Encodes the JObject and returns a JSON formatted string representing that JObject.
